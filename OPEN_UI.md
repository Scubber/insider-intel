# Open insider-intel UI

Servers should be on [http://127.0.0.1:5500/](http://127.0.0.1:5500/).

## Open inside Cursor (click one)

1. **[Open in Cursor Browser](command:simpleBrowser.show?%5B%22http%3A%2F%2F127.0.0.1%3A5500%2F%22%5D)** ← click this link
2. Or Command Palette (`Ctrl+Shift+P`) → **Simple Browser: Show** → paste `http://127.0.0.1:5500/`
3. Or Command Palette → **Browser: Open Integrated Browser** → same URL
4. Or Command Palette → **Tasks: Run Task** → **insider-intel: open in Cursor Browser**

> Tip: enable **Workbench › Browser: Open Localhost Links** so future `localhost` links open in the editor automatically.
